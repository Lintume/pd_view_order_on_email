== Description ==

pd_view_order_on_email plugin extends WooCommerce by adding a “Click here to go to the order page” link to customer order. Save time managing orders for your online store with this plugin. It is easy to install and there are no settings, just activate and enjoy! 

== Installation ==

1. Upload the entire `pd_view_order_on_email` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
